

## 介绍
     在校考试答题小程序，主要用于企业、事业单位、政府机构、学校和教育培训机构在线答题，在线模拟考试，在线刷题备考等场景。

+ 答题
+ 答题分单题模式和列表模式
+ 查看分数
+ 查看答案
+ 错题提醒
+ 查看答题历史记录
+ 查看错题记录
+ 生成海报
+ 选择题支持单选、多选
