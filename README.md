

## 介绍
    在线答题小博士小程序提供在线模拟考试，在线刷题，完全开源，从界面上相对简陋一些，如需优化版请联系开发者

## 如何体验

    微信小程序搜索：在线答题小博士

## 目前已完成功能
+ 答题
+ 答题分单题模式和列表模式
+ 查看分数
+ 查看答案
+ 错题提醒
+ 查看答题历史记录
+ 查看错题记录
+ 生成海报
